# Bản dịch Hytale tiếng Việt

Thư mục này chứa các tệp tin để dịch game Hytale sang tiếng Việt.

Cấu trúc thư mục:
- `client.lang`: Dịch giao diện người dùng (UI).
- `server.lang`: Dịch các vật phẩm (items), quái vật (mobs)...
- `avatarCustomization/`: Dịch các tùy chỉnh nhân vật.
- `meta.lang`: Thông tin về ngôn ngữ.
- `wordlists.lang`: Danh sách từ ngữ.
